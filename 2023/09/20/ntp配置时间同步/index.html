ntp配置时间同步

方法1：ntpd服务同步

  • 每台主机上安装yum -y install ntp
  • 配置/etc/ntp.conf配置文件
  • 启动ntpd服务

方法2：ntpd+ntpdate同步

  1. 选一台服务器作为ntp服务器

    安装yum -y install ntp

    启动ntpd服务

  2. 其他服务器作为客户端配置定时任务使用ntpdate同步时间

 yum install ntpdate
    echo "* * * * * /usr/sbin/ntpdate ntp服务器ip" >>/etc/crontab
    systemctl start ntpdate
    systemctl status ntpdate
    systemctl enable ntpdate
  1. 实例如下：
1、选择一台服务作为ntp服务器，安装ntp软件，并启动服务
    yum -y install ntp
    systemctl start ntpd
    systemctl enable ntpd
2、配置ntp客户端(客户端都要安装ntpdate)同步ntp服务器时间,下面命令都要在客户端执行
        yum install ntpdate
        echo "* * * * * /usr/sbin/ntpdate 172.172.31.32" >>/etc/crontab
        systemctl restart crond
        systemctl status crond
